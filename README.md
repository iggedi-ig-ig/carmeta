# Car Meta Hub
This repository contains a large collection of JSON-files.
All documents in this repository were generated using the same technique, 
which is _pretty good_ but not perfect, so expect some mistakes.
Also some countries / regions work better than others.

Feel free to leave a star if you find this useful :) 

## Q & A
> Why are some jsons zipped?

The maximum file size github allows is 100MB. If files are larger than that, zipping them usually makes them small enough.

> Why are some countries missing?

This depends on the country. Some countries I've just not generated yet, for some countries I can't map all types of cars yet,
and for some countries the json that I initially used to train the neural networks that are used in my tooling are gatekept, so I don't publish the results either.

> Why is there no US Json?

The United States are the one place where car meta is still not commonly known by geoguessr players, because it's not publically mapped.
I want this one thing to remain private "for me", because I spent so much time working on the tooling and I want it to have been "worth it".
Eventually I will release it, but I don't want to make any promises as to when, and ideally I don't want to be asked about it constantly either :)
